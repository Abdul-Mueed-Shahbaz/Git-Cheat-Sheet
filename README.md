# Git-Cheat-Sheet
# 1-> Creating a Repository
`git init  `   
`git remote add origin [repoLink]   `   
`git remote -v  `   
`git add .   `  
`git commit -m "your message"  `   
`git push origin [branchName]  `  
 # 2-> Creating a Pull Request
 Fork the desired the Repo  
`git clone [forkedRepoLink]`  
Do your Contribution
`git add .`  
`git commit -m "your message"`
`git push origin [branchName]`
